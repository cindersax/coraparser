# PDF Table Extractor API

The PDF Table Extractor API is a robust and versatile FastAPI application designed to extract tables from PDF documents, format them, and save the results in various file formats. It supports multiple table extraction methods and output formats, catering to a wide range of data processing needs.

## Features

- **Versatile Table Extraction**: Utilizes popular libraries like Camelot and Tabula for high-quality table extraction from PDFs.
- **Multiple Output Formats**: Supports saving extracted tables in formats like CSV and JSON, with the flexibility to add more.
- **FastAPI Framework**: Built on the modern, fast (high-performance) FastAPI framework, ensuring quick responses and asynchronous processing.
- **Easy Integration**: Designed for ease of integration with other applications and services via RESTful API endpoints.

## Getting Started

### Prerequisites

- Python 3.6+
- [FastAPI](https://fastapi.tiangolo.com/)
- [Camelot](https://camelot-py.readthedocs.io/en/master/)
- [Tabula](https://tabula.technology/)
